Industrial Energy Monitoring System

A full-stack industrial IoT solution that reads electrical variables from Schneider energy meters via BACnet, stores historical data in MySQL, and presents it through a real-time web dashboard for operational visibility and energy analysis.

CMI
Year
2024
Category
IIoT · Full-Stack · Energy
Role
Full-stack developer & system architect

// The Problem

The plant had Schneider energy meters installed across different electrical panels, but there was no centralized way to read, store, or visualize that data. Energy consumption tracking was manual, fragmented, and provided no visibility into consumption patterns or anomalies across the facility.

// The Solution

I designed and built a complete 3-layer IIoT solution from the protocol layer to the user interface — covering data acquisition, storage, and visualization in a single integrated platform.

// Architecture

L1
Acquisition — Python + BACnet/IP
Python service reads Schneider meters over BACnet protocol, polling electrical variables at regular intervals
L2
Storage — MySQL
All readings are persisted in a structured MySQL database for historical analysis and trend reporting
L3
Visualization — PHP + HTML + JS
Web dashboard queries the database and presents live readings, historical trends, and consumption reports

// Monitored Variables

The system captures all key electrical parameters from each meter in real time:

  • Voltage (line-to-neutral and line-to-line)
  • Current (A)
  • Active power (kW), Reactive power (kVAR), Apparent power (kVA)
  • Power factor
  • Frequency (Hz)
  • Active energy consumption (kWh) — cumulative and by period

// Dashboard Features

  • Real-time gauge and value display per meter
  • Monthly consumption summary by meter with totals
  • Historical trend charts for any variable
  • Scalable architecture — new meters can be added without code changes

// Results

  • Centralized energy visibility across the entire facility
  • Historical data available for energy efficiency analysis
  • Operational basis for detecting abnormal consumption patterns
  • Fully scalable to additional meters and facilities
