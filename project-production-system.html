Production Standardization System

End-to-end production management platform with live order tracking, KPI dashboards, and a route optimization algorithm that calculates the most efficient way to distribute production across manufacturing lines.

Client
Confidential (large enterprise)
Year
2024
Category
Software · Industry 4.0 · Microsoft Platform
Role
Developer & Automation Engineer

// The Problem

The client — a large enterprise manufacturing operation — lacked a unified system for creating, tracking, and reporting production orders. Each plant used its own methods, making it impossible to compare performance, standardize processes, or optimize how production was distributed across available lines.

// The Solution

I built a full-featured production management application using the Microsoft Power Platform, leveraging existing infrastructure the company already had (Microsoft 365 and SharePoint). The app was designed to cover the entire production lifecycle — from order creation to completion — with real-time visibility at every step.

// Route Optimization Algorithm

The most technically complex feature is the production route optimizer. Given a production plan, the algorithm evaluates the available manufacturing lines and calculates the optimal distribution of orders — minimizing idle time and maximizing throughput. Operators can run simulations before committing to a production schedule, comparing scenarios in real time.

// KPIs & Reporting

The platform tracks three core operational KPIs automatically: compliance (plan vs actual), assertiveness (forecast accuracy), and deviation (units off-target). Reports can be filtered by date range and exported. A dedicated reporting module covers order reports and efficiency reports by line.

// Technical Architecture

  • Canvas Power App as the main user interface across devices
  • SharePoint Lists as the operational database (orders, inventory, production records)
  • Power Automate flows for notifications, state transitions, and automated calculations
  • DAX formulas and custom logic for KPI computation inside the app
